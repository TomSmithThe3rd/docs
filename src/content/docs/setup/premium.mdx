**Engauge Premium** is a monthly subscription model that enables extra features
for your Engauge user and your Discord server.

For more information, visit the [main Premium page](https://engau.ge/premium).

## Profitability

Premium helps us run Engauge. It's a core value of Engauge to provide most
functionality for free, so we gate new features behind a Premium paywall for an
unspecified amount of time until the feature is stable.

Not all features are promised to become permanently available, but almost all
will make their way into the free feature space at some point.
